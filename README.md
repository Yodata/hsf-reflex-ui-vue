# hsf-reflex-ui-vue

This is the Vue UI for the REflex data sharing UI.  https://reflex-ui.yodata.me

UI dev/test and build happens in this repository.

Deploy to:
- developmnent: yodata/hsf-reflex-ui
- production: yodata/hsf-reflex-ui2

## install
```bash
$ npm install
```

## development
```bash
$ npm run dev
```

## build
```
$ npm run build
```
